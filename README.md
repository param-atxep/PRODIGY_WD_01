# PRODIGY_WD_01
## Interactive Navigation Menu

This project is developed as **Task-01** for the **Prodigy InfoTech Web Development Internship**.

The objective of this task is to create an **interactive navigation menu** that changes style when the user scrolls the page or hovers over menu items.

The navigation bar remains **fixed at the top of the page**, ensuring easy access to all sections while browsing the website.

---

## 🚀 Features

- Fixed Navigation Bar
- Hover Effects on Menu Items
- Scroll-based Navigation Color Change
- Smooth UI Animations
- Responsive Layout
- Clean and Modern Design

---

## 🛠 Technologies Used

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling and layout design  
- **JavaScript** – Interactivity and scroll effects

## 🎯 How It Works

1. The navigation menu is created using **HTML**.
2. **CSS** is used to style the navigation bar with a modern design and hover effects.
3. **JavaScript** detects when the user scrolls the page and changes the navigation bar style dynamically.
